
# IBM HR Analytics Excel Dashboard

## Overview

This project analyzes employee attrition data using Microsoft Excel. The dashboard provides HR insights through KPI metrics, Pivot Tables, Pivot Charts, and interactive slicers.

## Features

- Total Employees
- Attrition Count
- Attrition Rate
- Average Age
- Average Salary
- Attrition by Department
- Attrition by Gender
- Interactive Slicers

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Excel Formulas
- Slicers
